## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
   # Landing page cercas added to public/cercas/
3. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
4. Run the app:
